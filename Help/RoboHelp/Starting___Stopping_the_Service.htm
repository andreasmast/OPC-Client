Starting/Stopping the COM Server Wrapper

Simply open the "Services" icon in the Windows "Administrative Tools" and then locate the service, which is called "UA COM Server Wrapper". From there, you can configure the service behavior using the standard options provided by Windows, including starting and stopping the service.

Registering / Un-registering the Service

The following command line arguments can be used:

  • /install - installs the product as a Windows Service.

  • /uninstall - removes the product from the Windows Services.

  • /regserver - registers a COM product with the Windows Registry.

  • /unregserver - removes a COM product from the Windows Registry.
