COM Interop Components MergeModule

Overview

For installing the UA/COM Proxy/Wrapper services that are provided by the OPC Foundation.

Deployed Components

  • UA COM Proxy Server
  • UA COM Server Wrapper
  • Configuration files

Installation Locations

  • Program Files > OPC Foundation > Unified Architecture > ComInterop:
    • interop.NetFwTypeLib.dll
    • UA Client Library (Opc.Ua.Client.dll)
    • UA COM Interop Library (Opc.Ua.ComInterop.dll)
    • Opc.Ua.ComProxyServer.Config.xml
    • Opc.Ua.ComProxyServer.exe.config
    • Opc.Ua.ComServerWrapper.Config.xml
    • Opc.Ua.ComServerWrapper.exe.config
    • UA Configuration Library (Opc.Ua.Configuration.dll)
    • UA Core Library (Opc.Ua.Core.dll)
    • UA Server Library (Opc.Ua.Server.dll)
    • UA COM Proxy Server
    • UA COM Server Wrapper
  • Program Files > Reference Assemblies > OPC Foundation > Unified Architecture > v1.1:
    • UA COM Interop Library
    • UA COM Interop Library (documentation)
