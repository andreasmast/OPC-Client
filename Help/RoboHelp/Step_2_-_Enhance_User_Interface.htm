Step 2 - Enhancing the User Interface

In this section we will simply add a new menu item to the "Server" menu called "Subscribe" available at the top of the window. This option will subscribe to the selected node and will display the value changes in a new window.

This step is a continuation from Step 1 - New Project.

Creating the New Window

  1. Right-click on the project and choose "Add -> Windows Form" from the menu.

  2. Name the window "SubscriptionOutput".

  3. The window designer will appear and we will now add a label to the form that will occupy the entire window space:

    1. From the Visual Studio .NET 2008 "Common Controls" toolbox double-click the Label control, which will add it to the form.

    2. Select the new label by clicking on it once.

    3. In the property-grid you will:

      1. Select the Dock property and specify the value as Fill.

      2. Change the AutoSize property to False.

      3. Change the Modifiers property to Protected Internal.

    4. Our form is now ready for use.

Modifying the Menu

  1. Double-click on MainForm.cs to launch the window designer.

  2. At the top of the window you will see the main application menu, click on the Server menu and the options will appear below it.

  3. In the bottom box (which shows "Type Here") type in the name "Subscribe".
