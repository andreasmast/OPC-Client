Local Discovery Server (LDS) Overview

The UA Specifications Part 4 - Services (http://www.opcfoundation.org/ua/part4) and UA Specifications Part 12 - Discovery (http://www.opcfoundation.org/ua/part12) describe the purpose and operations of a local discovery server. The short explanation is that this type of server merely provides a list of available OPC UA applications to any application that queries it; where each application is running and registering (frequently) with the LDS.
