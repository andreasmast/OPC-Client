Developing UA Applications

You can develop your own Client and Server applications that support OPC Unified Architecture technology.

The OPC Foundation Unified Architecture Technology Samples are built upon the UA .NET API which is available to OPC Foundation Corporate Members. Source code of the applications documented in this help file is provided, if you have downloaded the installer containing source code.

Many commercial OPC UA development SDKs/Toolkits are available. The OPC Foundation strongly encourages the use of Certified commercial development products. The list of Lab Certified products are available online at: http://www.opcfoundation.org/certifiedproducts
